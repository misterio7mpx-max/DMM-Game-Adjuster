導入方法
ブラウザ拡張機能「Tampermonkey」をインストールします。

新規スクリプトを作成し、本スクリプトのコードを貼り付けて保存します。

対象のゲームページ（https://play.games.dmm.co.jp/game/*）を開くと適用されます。

操作方法

画面レイアウトの完全制御
ゲームフレームの拡大・縮小（Zoom）、X/Y座標のオフセット調整をスライダーで直感的に変更可能です。右側の不要な領域を隠す「目隠し（Mask）」機能も備えています。

観察モード（ズーム＆ロック）の開始: 画面右側の「黒い目隠し領域」を左クリックすると、即座に設定した倍率でのズームが始まり、画面がロック状態（観察モード）になります。

エフェクトの発生: 観察モード中にゲーム画面を左クリックすると、巨大なハートと波紋のエフェクトが発生し、設定された効果音が再生されます。

観察モードの解除: 画面のどこかを右クリックするか、再度「黒い目隠し領域」を左クリックすることで、瞬時に通常のゲーム操作画面に戻ります。

設定パネル: 右上の「⚙」ボタンでパネルを開きます。パネル以外の暗い背景部分をクリックするとパネルを閉じます。

音声（効果音）の設定方法
本スクリプトには音声ファイル自体は含まれていません。以下の手順で設定してください。

ファイルの用意: 再生したい音声ファイル（MP3推奨）を準備します。

サーバーの起動: ローカル環境（PC内）でサーバーを立てるか、外部のストレージサービスにファイルをアップロードし、その直リンクURLを取得します。

デフォルトでは http://localhost:8000/heart.mp3 が設定されています。

URLの入力: ⚙設定パネル内の「効果音URL」欄に、取得したURLを貼り付けます。

無音化（ミュート）: 音を鳴らしたくない場合は、この入力欄を「空欄」にしてください。

UIとカスタマイズ
XY軸オフセット: ズームした際、マウス位置から映像をどれだけ上下左右にずらすかを±1200pxの範囲で調整できます。

巨大ハート: ハートの基本サイズを従来の3倍に拡大しています。ランダム幅を調整することで画面全体を覆うような演出も可能です。
