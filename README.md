## 概要 (Overview)
ブラウザゲームプラットフォーム（DMM Games）のプレイ画面を最適化し、誤操作防止機能や独自の視覚・聴覚エフェクトを追加するTampermonkey向けユーザースクリプトです。
不要なヘッダーやナビゲーションの非表示、画面の拡大縮小・位置調整に加え、「緊急回避（パニックボタン）」機能を搭載しています。

## 主な機能 (Features)
* **🎮 画面レイアウトの完全制御**
  ゲームフレームの拡大・縮小（Zoom）、X/Y座標のオフセット調整をスライダーで直感的に変更可能です。右側の不要な領域を隠す「目隠し（Mask）」機能も備えています。
* **🔒 クリックガード（ロック）機能**
  専用のカスタムカーソル（人差し指シルエット）を表示し、ゲーム画面への誤クリックを防止する透明なオーバーレイを展開します。
* **✨ インタラクティブな視覚・聴覚エフェクト**
  ロック画面クリック時、CSSアニメーションによる「光の波紋（リップル）」と、ランダムなサイズ・位置で浮かび上がる「ハート」のエフェクトを描画します。同時にローカルサーバーから効果音を再生します。
* **🙈 緊急回避（パニックボタン）**
  指定した画像（ローカルURL等）を瞬時に画面全体に被せ、ゲーム画面を隠蔽する緊急オーバーレイを展開します。画像の縮尺や表示位置も細かく調整可能です。
* **⚙️ 動的コントロールパネル**
  すべての設定値は画面右上の歯車アイコンから展開するパネルで調整可能であり、`localStorage` に自動保存されます。

## 動作環境 (Requirements)
* Tampermonkey などのユーザースクリプトマネージャー
* 効果音や緊急画像をローカルから読み込むためのローカルサーバー（例: `localhost:8000`）の稼働
* 音声再生が許可されたブラウザ環境

## インストールと設定 (Setup)
1. スクリプトマネージャーに本コードを追加します。
2. 対象のゲームページを開くと、画面右側にコントロール用の丸いボタンパネルが表示されます。
3. `⚙` (歯車ボタン) を押すと、画面レイアウトやエフェクトの詳細設定パネルが開きます。
4. 各種スライダーで「拡大縮小」「左右位置」「ハートのランダム幅」などを調整してください。設定は自動的に保存されます。
5. `🔓` (ロックボタン) を押すとクリックガードが有効になり、画面端のガイド表示とともに、クリック時の波紋・ハートエフェクトが起動します。
6. `🙈` (緊急回避ボタン) を押すと、瞬時に指定した緊急画像が全画面を覆います。
